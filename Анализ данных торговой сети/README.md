## Анализ данных торговой сети

На базе обезличенных исторических данных сети торговых точек выполнила комплексный анализ операционных процессов. 

В результате:
* выявила критические инциденты, влияющие на эффективность работы;
* сформулировала рекомендации по устранению проблем и оптимизации ключевых процессов.

### Ключевые направления анализа:
1. Динамика транзакций после запуска новой торговой точки.
2. Нагрузка на серверную инфраструктуру.
3. Функционирование касс самообслуживания.
4. Эффективность проведённой маркетинговой акции.

Отчеты: 
1. Анализ аномалий длительности транзакций 
2. Анализ работы касс самообслуживания, анализ маркетинговой акции
