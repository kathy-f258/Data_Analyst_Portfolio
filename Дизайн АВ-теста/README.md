# Анализ результатов А/В тестирования для определения влияния нового вида уведомлений

[Ссылка на тетрадь](Дизайн АВ-теста/AB_test_push_Фоломеева.ipynb)

## Описание проекта
Сравнение эффективности пуш‑уведомлений и баннеров в мобильном приложении ритейлера.

## Цель исследования 
Оценить, какой канал маркетинговых коммуникаций — пуш‑уведомления или баннеры в приложении — эффективнее влияет на ключевые бизнес‑метрики:
  * конверсию из уведомления в покупку;
  * средний чек покупки.

## Обоснование
До эксперимента основным инструментом информирования клиентов о акциях и скидках были баннеры внутри приложения. Отдел маркетинга выдвинул гипотезу, что пуш‑уведомления (всплывающие сообщения на экране смартфона, даже когда приложение закрыто) могут:
  * лучше привлекать внимание;
  * повышать вовлечённость;
  * стимулировать покупки активнее, чем статичные баннеры.

## Методология
_Дизайн теста:_ классическое A/B‑тестирование.

_Группы:_
* контрольная (баннеры) — получала уведомления о акциях и скидках в виде баннеров в приложении
* тестовая (пуш‑уведомления) — получала аналогичные сообщения в формате пуш‑уведомлений

_Сплит:_ клиенты разделены на две равные группы случайным образом.

_Длительность:_ 3 месяца.

_Таргет‑метрики:_
  * конверсия из уведомления в покупку (конверсия= число полученных уведомлений / число покупок после уведомления×100%)
  * средний чек покупки (средний чек= число покупок / общая сумма покупок)

_Контроль внешних факторов:_ 
  * идентичность контента уведомлений (акции и скидки одни и те же)
  * синхронизация временных периодов для обеих групп
  * исключение влияния сезонности и масштабных промо‑акций вне теста

## Полученные результаты:
 * проведен EDA
 * выполнена сегментация по городам и торговым точкам и рассчитаны результаты A/B-теста на каждом сегменте, сделана визуализация
 * АВ-тест показал, что конверсии между тестовой и контрольной группами статистически различаются — эксперимент оказал реальное влияние на поведение клиентов:
      - пуш‑уведомления повышают конверсию по сравнению с баннерами
      - пуш‑уведомления оказывают влияние на увеличение среднего чека
 * результаты исследования выгружаются в форме удобного Excel-отчета

 
## Значение для бизнеса
Результаты теста позволят:
  * оптимизировать маркетинговый бюджет (перераспределить ресурсы в пользу более эффективного канала)
  * повысить общую конверсию и выручку за счёт точечного воздействия на клиентов
  * сформировать базу для дальнейших экспериментов с персонализацией уведомлений
